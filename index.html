How to play

Use arrow keys to move tiles on map

Game idea

There is red and blue armies represented by tiles with numbers.

It's notimportant with army you'll support. The important thing is that one will have much more score than the other.

Numbers on tiles represent army strength

How to fight

If two tiles moves one each other they will fight.

[4] > [1] = [3] One with bigger number will win (and lose 1point for this action).

[2] + [2] = die If they have same number both will lose.

Blank tile

[ ] Blank/wall tile that is just a simple obstacle.

Bonuses tiles

[!] + [4] = die The yellow tile will destroy any tile.

[+] + [1] = [2] The green tile will add 1 point to any army.
